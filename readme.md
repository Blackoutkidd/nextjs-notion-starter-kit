# ✈️ USAF Separation Guide

A chronological roadmap for Airmen transitioning to civilian life. Built with Notion and optimized for mobile to help veterans navigate the 24-month transition window.

### 📱 Live Mobile App
[Access the Separation Checklist Here](https://separation-checklist.notionapps.com)

### 🛠️ Tech Stack
* **CMS & Database:** Notion
* **Mobile Interface:** NotionApps (PWA)
* **Web Framework:** Next.js (Deployed via Vercel)
* **Content:** AI-assisted via NotebookLM

### 🏃‍♂️ About the Project
Developed by a USAF Veteran and current MS in Data Science student. This project is dedicated to streamlining the military-to-civilian transition.
